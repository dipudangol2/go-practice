# go-practice
Programs written while learning Go.

## Programs

| Directory | Topic |
|-----------|-------|
| [`hello/`](hello/) | Hello, World! |
| [`variables/`](variables/) | Variables, constants, and zero values |
| [`control-flow/`](control-flow/) | if/else, for loops, and switch statements |
| [`functions/`](functions/) | Functions, multiple returns, variadic, closures |
| [`arrays-slices/`](arrays-slices/) | Arrays, slices, make, copy, 2D slices |
| [`maps/`](maps/) | Maps, CRUD operations, iteration |
| [`structs/`](structs/) | Structs, methods, embedding |
| [`interfaces/`](interfaces/) | Interfaces, type assertions, type switches |
| [`error-handling/`](error-handling/) | Errors, custom types, wrapping, defer/recover |
| [`goroutines/`](goroutines/) | Goroutines, channels, WaitGroup, Mutex |

## Running a program

```bash
go run ./<directory>/main.go
```

For example:

```bash
go run ./hello/main.go
```
